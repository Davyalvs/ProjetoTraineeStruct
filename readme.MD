Esse projeto foi criado no decorrer do processo trainee da empresa junior {Struct} como forma de aprendizado dos seguintes tópicos:

-   Git e Github
-   Banco de dados
-   Figma
-   HTML e CSS
-   Typescript
-   React
-   Nextjs
-   Prisma
-   APIs
-   NextAuth
